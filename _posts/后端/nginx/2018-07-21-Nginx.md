#1 Nginx概述
## 1.1 nginx配置
### 静态文件服务器

```
server {
    listen  80;
    server_name    172.19.7.60;
    charset utf-8;
    root /data/FileService;
    location / {
        autoindex on; # 索引
        autoindex_exact_size on; # 显示文件大小
        autoindex_localtime on; # 显示文件时间
    }
}
```
